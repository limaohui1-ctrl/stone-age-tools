## 🛠️ 使用方法

### Linux用户
```bash
# 直接运行
./工具名

# 查看帮助
./工具名 --help

# 示例：运行网络稳定性工具包主程序
./核心工具/主程序 --help
```

### Windows用户
1. **重命名为.exe后缀**
   ```bash
   mv 工具名 工具名.exe
   ```

2. **使用Wine运行**
   ```bash
   wine 工具名.exe
   ```

3. **或在Windows上重新编译**
   ```bash
   pip install pyinstaller
   pyinstaller --onefile 工具名.py
   ```

## ⚠️ 注意事项

### 常见问题
1. **无法运行**: 尝试以管理员身份运行
2. **文件较大**: EXE文件包含Python运行时，这是正常现象
3. **防病毒软件**: 可能被误报，请添加到白名单
4. **权限问题**: 使用 `chmod +x 工具名` 添加执行权限

### 系统要求
- **Linux**: 大多数现代Linux发行版
- **Windows**: 需要Wine或重命名为.exe
- **内存**: 至少1GB可用内存
- **磁盘**: 至少100MB可用空间
