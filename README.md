# 🚀 石器时代工具打包启动包

专为石器时代游戏脚本工具打包设计的Windows启动包，提供完整的自动化打包流程。

## 📋 快速开始

### 第一步：环境准备
1. **下载启动包**: 将此文件夹复制到Windows电脑
2. **运行环境设置**: 双击 `01_setup.bat`
3. **管理员权限**: 选择"以管理员身份运行"

### 第二步：项目下载
1. **下载项目文件**: 双击 `02_download.bat`
2. **选择下载方式**: 
   - 推荐: 从GitHub克隆 (需要Git)
   - 备用: 下载ZIP文件
   - 本地: 使用已有文件

### 第三步：开始打包
1. **选择打包模式**: 双击 `03_pack.bat`
2. **推荐模式**:
   - 🧪 **测试模式**: 打包1个简单工具 (首次使用)
   - 📦 **批量模式**: 打包前3个核心工具
   - 🚀 **完整模式**: 打包所有10个工具

## 🛠️ 问题解决

### 常见问题

#### Q1: 管理员权限错误
**问题**: "需要管理员权限"
**解决**: 右键点击BAT文件 → "以管理员身份运行"

#### Q2: Python安装失败
**问题**: "Python未安装"
**解决**: 
1. 手动下载Python 3.8.10
2. 安装时勾选"Add Python to PATH"
3. 重新运行脚本

#### Q3: 网络连接问题
**问题**: 下载失败或超时
**解决**:
1. 检查网络连接
2. 使用VPN (如果需要)
3. 选择ZIP下载方式

#### Q4: 依赖安装失败
**问题**: "pip install"失败
**解决**:
1. 使用国内镜像源: `pip install -i https://pypi.tuna.tsinghua.edu.cn/simple`
2. 手动安装核心依赖
3. 跳过非关键依赖

#### Q5: 打包失败
**问题**: PyInstaller错误
**解决**:
1. 检查Python版本 (需要3.6+)
2. 清理缓存: `pyinstaller --clean`
3. 简化打包命令

### 错误日志位置
- **环境验证日志**: `check_env.py` 输出
- **打包错误日志**: `logs/` 目录
- **系统错误**: 查看命令提示符输出
