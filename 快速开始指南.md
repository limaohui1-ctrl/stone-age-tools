# 🚀 石器时代工具打包 - 快速开始指南

## 📋 项目概述

**石器时代工具打包项目**旨在将10个核心石器时代脚本工具打包为Windows EXE文件，方便用户直接使用。

### 包含的工具
1. 🎯 **网络稳定性增强工具包** (5个工具)
2. 🔧 **脚本开发工具** (3个工具)
3. 🔤 **验证码识别工具** (1个工具)
4. 📍 **坐标管理系统** (1个工具)

## 🪟 Windows环境快速开始

### 第一步：下载项目
```bash
# 克隆仓库
git clone https://github.com/limaohui1-ctrl/stone-age-tools.git

# 或下载ZIP文件
# 从GitHub页面下载ZIP并解压
```

### 第二步：环境准备
1. **进入项目目录**
   ```bash
   cd stone-age-tools
   ```

2. **运行环境设置脚本**
   - 双击 `start_packing.bat`
   - 选择"以管理员身份运行"
   - 按照提示完成环境配置

3. **自动安装内容**
   - ✅ Python 3.8.10 (如果未安装)
   - ✅ PyInstaller (打包工具)
   - ✅ Git (版本控制)
   - ✅ 所有依赖库

### 第三步：开始打包

#### 选项1: 测试模式 (推荐首次使用)

# 打包最简单的网络波动检测器
# 验证打包流程
```

#### 选项2: 批量模式

# 1. 网络波动检测器
# 2. 坐标管理系统
# 3. 脚本调试器
```

#### 选项3: 完整模式
```bash
# 运行打包脚本
03_pack.bat

# 选择选项3: 完整模式
# 打包所有10个工具
# 预计时间: 2-3小时
```

## 🛠️ 技术支持

### 常见问题解决

#### Q1: 管理员权限错误
**问题**: "需要管理员权限"
**解决**: 右键点击BAT文件 → "以管理员身份运行"

#### Q2: Python安装失败
**问题**: "Python未安装"
**解决**: 
1. 手动下载Python 3.8.10
2. 安装时勾选"Add Python to PATH"
3. 重新运行脚本

#### Q3: 依赖安装失败
**问题**: "pip install"失败
**解决**: 使用国内镜像源
```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt
```

#### Q4: 打包失败
**问题**: PyInstaller错误
**解决**:
1. 清理缓存: `pyinstaller --clean`
2. 简化命令: `pyinstaller --onefile script.py`
3. 检查Python版本: 需要3.6+

### 错误日志位置
- **环境验证日志**: `check_env.py` 输出
- **打包错误日志**: `logs/` 目录
- **系统错误**: 查看命令提示符输出

## 📊 打包优化建议

### 文件大小优化
```bash
# 使用UPX压缩 (减小EXE大小)
pyinstaller --onefile --upx-dir=/path/to/upx script.py

# 排除不必要的模块
pyinstaller --onefile --exclude-module=unnecessary script.py
```

### 性能优化
```bash
# 使用单文件模式
pyinstaller --onefile script.py

# 禁用控制台窗口 (GUI程序)
pyinstaller --windowed script.py

# 优化启动速度
pyinstaller --onefile --noupx script.py
```
